-------------------------------Comandos git-------------------------------

-Clonar repositório remoto na maquina atual
git clone https://github.com/ahagemeier7/Estrutura-de-dados24.git

-Ver status do repositorio loal
git status 

-Adicionar arquivos não rastreados no git
git add .

-criar ponto marcação no projeto
git commit -m escreveamensagem

-Verificar repositorios remotos configurados no repo local
git remote -v

-Enviar comits pendentes para o repo remoto
git push origin main

git checkout -b "NomeDaBranch" //Cria Branch nova

git checkout NomeBranch // Troca a Branch

pit push origin NomeDaBranch
pit pull origin NomeDaBranch
git remote -v mostra repositórios Remotos
